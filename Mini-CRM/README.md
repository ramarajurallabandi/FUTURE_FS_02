# 📘 Client Lead Management System (Mini CRM)

📌 **Internship Task – Future Interns**  
**Task 2: Client Lead Management System (Mini CRM)**

This project is developed as part of the **Future Interns – Full Stack Web Development Internship**.  
The Mini CRM helps manage client leads generated from contact forms by providing essential CRM functionalities such as lead creation, validation, listing, and deletion.

## 🚀 Project Overview

The **Mini CRM** is a full-stack web application that allows users to:

- Add new client leads  
- View all leads in a dashboard  
- Delete leads instantly  
- Validate user input before submission  

This project demonstrates **CRUD operations**, **REST API development**, and **MongoDB database integration**.

## ✨ Key Features

- ➕ Add new leads (Name, Email, Phone, Address)  
- 📋 View all leads in dashboard  
- 🗑️ Delete leads without page reload  
- ✅ Field-level validation for all inputs  
- 📧 Email domain validation  
- 📱 Phone number validation (numeric only, exactly 10 digits)  
- 🗄️ MongoDB database integration  
- 🌐 RESTful APIs using Express.js  

---

## 🌐 Using the Application

- Open **test.html** → Add new client leads  
- Open **dashboard.html** → View and delete leads  

✔ No Postman required  
✔ Simple browser-based interaction  

---

## 🔍 API Endpoints

| Method | Endpoint | Description |
|-------|----------|-------------|
| POST  | `/api/leads` | Add a new lead |
| GET   | `/api/leads` | Retrieve all leads |
| DELETE | `/api/leads/:id` | Delete a lead |
